# CRE2003 Source Code — README

> Archival copy of the original `README.TXT` shipped with the CRE2003 source-code
> diskette. Filenames that previously appeared as broken Markdown links (e.g.
> `[CREX-S.ZIP](http://CREX-S.ZIP)`) have been restored to plain filenames; the
> wording is otherwise unchanged.

## Files included with your CRE2003 source code

| File           | Description                                              |
| -------------- | -------------------------------------------------------- |
| `CREX-S.ZIP`   | All forms, program files, etc. (DLLs, OCXs **NOT** included) |
| `CREX-D.ZIP`   | Main database file                                       |
| `PKUNZIP.EXE`  | PKUNZIP decompression utility                            |
| `README.TXT`   | Cash Register Express source-code instructions           |

`X` = version number (displayed on the diskette).

> **Example:** if the release is 5.01, the files are `CRE501-S.ZIP` and
> `CRE501-D.ZIP`.

## Building the executable

`CRE2003.EXE` is **not** included with the source code, but can be built using
**Microsoft Visual Basic 6.0 Professional Edition**, or by installing a fully
working copy of CRE 2003.

## Unzipping

The distribution files are compressed with PKZIP. Use `PKUNZIP.EXE` to
decompress them. For help, run:

```
Cash Register Express /?
```

at the MS-DOS prompt.

## Where files go

| Archive        | Destination directory   |
| -------------- | ----------------------- |
| `CREX-S.ZIP`   | `\CRE2000`              |
| `DATX-S.ZIP`   | `\CRE2000\DATA2000`     |

## Required runtime components

To obtain all the necessary DLL and OCX files, install a fully running copy of
**Microsoft Visual Basic 6.0 Professional Edition** as well as a fully running
copy of **CRE 2003**.
